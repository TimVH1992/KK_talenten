# Talentontwikkeling

Applicatie om leerlingen automatisch te verdelen over talentenperiodes op basis van hun 1e, 2e en 3e voorkeur, capaciteit en eerdere deelnames.

## Doel
- Keuzes per leerling registreren
- Talenten per periode beheren
- Automatische verdeling uitvoeren
- Manuele aanpassingen mogelijk maken
- Historiek bewaren
