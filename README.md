# Talentontwikkeling

Backend-leerproject voor het beheren van talentontwikkeling binnen een school.

De applicatie verdeelt leerlingen automatisch over ingerichte talenten op basis van hun 1e, 2e en 3e voorkeur, capaciteit en eerdere deelnames. Nadien kan de talentcoördinator de verdeling manueel aanpassen.

## Doelen

- Java en OOP toepassen in een realistische context
- Domeinmodel en businessregels correct uitwerken
- Repository Pattern en JDBC gebruiken
- Oracle Database integreren
- Project opbouwen als portfolio voor backend development
